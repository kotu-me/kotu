# Kotu — Official SDKs & Developer Tools

Kotu is a modern, developer-first link platform offering link management, analytics, deep linking, open-in-app features, and more — designed to be fast, secure, and globally scalable.

This repository hosts the official SDK implementations for multiple languages.

## Available SDKs

Kotu provides first-class libraries for:

### JavaScript / NodeJS  
NPM Package:  
**https://www.npmjs.com/package/kotu**

### Python  
PyPI Package:  
**https://pypi.org/project/kotu**

### Go  
Go Module:  
**https://pkg.go.dev/kotu**

All SDKs follow the same API structure and are built to deliver a consistent and intuitive developer experience across languages.
